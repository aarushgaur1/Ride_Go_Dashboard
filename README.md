## 📖 Project Overview

The **RideGo Data Analysis Dashboard** is an end-to-end data analytics project built using **SQL** and **Power BI**. The project analyzes ride booking data to uncover trends in bookings, revenue, cancellations, vehicle performance, and customer satisfaction. It provides interactive dashboards that help stakeholders make informed, data-driven decisions.

---

## 🎯 Project Objectives

- Analyze booking and revenue trends.
- Monitor ride cancellations and identify their reasons.
- Compare vehicle type performance.
- Evaluate payment method usage.
- Track customer and driver ratings.
- Create an interactive dashboard for business insights.

---

## 🛠️ Tech Stack

The project was built using the following technologies:

- 📊 **Power BI Desktop** – Dashboard development and data visualization.
- 🗄️ **MySQL** – Data extraction and analysis using SQL queries.
- 📐 **DAX** – Calculated measures and KPIs.
- 🔄 **Power Query** – Data cleaning and transformation.
- 📁 **Excel/CSV Dataset** – Source data.

---

## 📊 Dashboard Pages

### 📌 Overall Dashboard
Provides a high-level overview of the business including:
- Total Bookings
- Total Booking Value
- Booking Status Distribution
- Ride Volume Trend
- Date Filter

### 🚗 Vehicle Type Dashboard
Compares the performance of different vehicle categories:
- Total Booking Value
- Successful Booking Value
- Average Distance Travelled
- Total Distance Travelled

### 💰 Revenue Dashboard
Analyzes revenue performance through:
- Revenue by Payment Method
- Top 5 Customers
- Payment Method Distribution

### ❌ Cancellation Dashboard
Provides insights into ride cancellations:
- Total Bookings
- Successful Bookings
- Cancelled Bookings
- Cancellation Rate
- Customer Cancellation Reasons
- Driver Cancellation Reasons

### ⭐ Ratings Dashboard
Shows service quality across vehicle types:
- Average Driver Rating
- Average Customer Rating

---

## 📈 Key Insights

- Approximately **62%** of bookings were completed successfully.
- The overall cancellation rate is around **28%**.
- Cash and UPI are the most frequently used payment methods.
- Prime Sedan generated the highest booking value.
- Average customer and driver ratings remained close to **4.0** across all vehicle types.
